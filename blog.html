Important FAQ

What is document object model (DOM)

DOM is a programming interface that represents HTML or XML documents as a tree structure. Each element, attribute, and piece of text in the document is a node, allowing JavaScript to interact with and manipulate the document

How do we select an element.

There are 4 ways to select an element
1.getElementById
2.getElementsByClassName
3.getElementsByTagName
4.querySelector

What is event delegation in the context of DOM?

Event delegation is a technique that involves attaching a single event listener to a parent element instead of individual child elements. When an event occurs, it bubbles up to the parent, allowing you to check which child triggered it.

How do we manipulate an elements attributes and styles with the DOM?

1. setAttribute
2. getAttribute
3. removeAttribute
